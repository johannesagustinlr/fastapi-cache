# fastapi-cache


```bash
git clone git@github.com:johannesagustinlr/fastapi-cache.git
```

```bash
cd fastapi-cache
```

```bash
docker-compose up --build
```
to send data

```bash
localhost:8000/payload 
```
example payload

```bash
{
    "list_1":[
        "list1.1",
        "list1.2",
        "list1.3"
    ],
    "list_2":[
        "list2.1",
        "list2.2",
        "list2.3"
    ]
}
```

to get data

```bash
localhost:8000/{id} 
```
